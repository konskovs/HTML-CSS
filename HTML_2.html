1. Internal CSS

Koncepcija: Stilius <head> dalyje (<style>).
Uzduotis: Istrinkite visus inline stilius. Sukurkite <style> bloka head dalyje. Nustatykite, kad visi p elementai butu pilkos spalvos (gray).
Testas: Visi paragrafai puslapyje turi tapti pilki.

Mano puslapis su stiliumi

Sitas tekstas dabar yra pilkas.

Ir sitas taip pat yra pilkas!

CSS
<style>
  p {
    color: gray;
  }
</style>
HTML
<h2>Mano puslapis su stiliumi</h2>
<p>Sitas tekstas dabar yra pilkas.</p>
<p>Ir sitas taip pat yra pilkas!</p>

2. Klases selektorius

Koncepcija: Klases (.klases-pavadinimas).
Uzduotis: Sukurkite tris paragrafus. Pirmam ir treciam suteikite klase "svarbus". CSS dalyje nustatykite, kad .svarbus tekstas butu paryskintas (font-weight: bold).
Testas: Pirmas ir trecias paragrafai paryskinti, antras – paprastas.

Svarbus tekstas (Pirmas)

Paprastas tekstas (Antras)

Vel svarbus tekstas (Trecias)

CSS
.svarbus {
  font-weight: bold;
}
HTML
<p class="svarbus">Svarbus tekstas (Pirmas)</p>
<p>Paprastas tekstas (Antras)</p>
<p class="svarbus">Vel svarbus tekstas (Trecias)</p>

3. ID selektorius

Koncepcija: Unikalus identifikatorius (#id-pavadinimas).
Uzduotis: Sukurkite viena antraste su id "pagrindinis". CSS dalyje nustatykite siam ID unikalia spalva (pvz., purple).
Testas: Tik sis elementas turi buti violetinis.

Unikali antraste

Paprastas tekstas po ja.

CSS
#pagrindinis {
  color: purple;
}
HTML
<h2 id="pagrindinis">Unikali antraste</h2>
<p>Paprastas tekstas po ja.</p>

4. Srifto dydis ir tipas

Koncepcija: font-family, font-size.
Uzduotis: Visam body elementui nustatykite srifta Arial. Antrastems h1 nustatykite dydi 40px.
Testas: Tekstas turi pasikeisti i Arial, o antraste tapti didesne.

Didele antraste

Sitas tekstas dabar turetu buti Arial sriftu.

CSS
body {
  font-family: Arial;
}

h1 {
  font-size: 40px;
}
HTML
<h1>Didele antraste</h1>
<p>Sitas tekstas dabar turetu buti Arial sriftu.</p>

5. Div konteineris

Koncepcija: Grupavimas (<div>).
Uzduotis: Idekite antraste ir paragrafa i <div> elementa. Suteikite div'ui klase "kortele" ir nustatykite sviesiai pilka fona.
Testas: Turi matytis pilkas blokas, kurio viduje yra tekstas.

Be vidiniu tarpu

Tekstas lieciasi prie krastu.

CSS
.kortele {
  background-color: #f0f0f0;
}
HTML
<div class="kortele">
  <h3>Be vidiniu tarpu</h3>
  <p>Tekstas lieciasi prie krastu.</p>
</div>

6. Krastines (Borders)

Koncepcija: border.
Uzduotis: Sukurkite mygtuka. CSS dalyje uzdekite jam 2px storio, vientisa (solid), juoda remeli.
Testas: Aplink mygtuka turi buti ryskus remelis.
CSS
.remelis-mygtukas {
  border: 2px solid black;
  padding: 10px 20px;
  background-color: white;
}
HTML
<button class="remelis-mygtukas">Mano Mygtukas</button>

7. Vidiniai tarpai (Padding)

Koncepcija: padding (tarpas tarp turinio ir remelio).
Uzduotis: Paimkite anksciau sukurta kortele (div). Pridekite padding: 20px.
Testas: Tekstas neturi liestis prie pilko fono krastu.

Kortele su padding

Tekstas dabar turi vietos kvepuoti!

CSS
.su-padding {
  background-color: #f0f0f0;
  padding: 20px;
}
HTML
<div class="su-padding">
  <h3>Kortele su padding</h3>
  <p>Tekstas dabar turi vietos kvepuoti!</p>
</div>

8. Isoriniai tarpai (Margin)

Koncepcija: margin (tarpas tarp elementu).
Uzduotis: Sukurkite du atskirus div kvadratus. Pridekite jiems margin-bottom: 20px.
Testas: Kvadratai neturi buti sulipe.
1
2
CSS
.kvadratas {
  width: 50px;
  height: 50px;
  background-color: orange;
  margin-bottom: 20px;
}
HTML
<div class="kvadratas">1</div>
<div class="kvadratas">2</div>

Koncepcija: width, height.

10. Uzapvalinti kampai

Koncepcija: border-radius.
Uzduotis: Paimkite zalia kvadrata ir nustatykite border-radius: 10px.

CSS
.su-kampais {

<div class="deze su-kampais"></div>

11. Nuorodos be pabraukimo

Koncepcija: text-decoration.
Uzduotis: Sukurkite nuoroda. CSS dalyje nurodykite text-decoration: none ir pakeiskite spalva i juoda.
Testas: Nuoroda turi atrodyti kaip paprastas tekstas (be melyno pabraukimo).
As esu nuoroda
CSS
.svari-nuoroda {
  text-decoration: none;
  color: black;
}
HTML
<a href="#" class="svari-nuoroda">As esu nuoroda</a>

12. Pelytes kursorius

Koncepcija: cursor.
Uzduotis: Sukurkite elementa (pvz., span), kuris atrodo kaip mygtukas. Nustatykite cursor: pointer.
Testas: Uzvedus pele ant elemento, rodikle turi pasikeisti i rankyte (rodyti, kad galima spausti).
Uzeik pele ant manes
CSS
.span-mygtukas {
  cursor: pointer;
  background-color: #f0f0f0;
  padding: 10px 20px;
  border: 1px solid #999;
}
HTML
<span class="span-mygtukas">Uzeik pele ant manes</span>

13. Lenteles (Table) remeliai

Koncepcija: table, border-collapse.
Uzduotis: Sukurkite paprasta 2x2 lentele. CSS nustatykite border: 1px solid black lentelei ir td. Naudokite border-collapse: collapse.
Testas: Turi matytis tvarkinga lentele su viengubais remeliais.
1 Celė 2 Celė
3 Celė 4 Celė
CSS
table {
  border-collapse: collapse;
}

td {
  border: 1px solid black;
  padding: 10px;
}
HTML
<table>
  <tr>
    <td>1 Cele</td>
    <td>2 Cele</td>
  </tr>
  <tr>
    <td>3 Cele</td>
    <td>4 Cele</td>
  </tr>
</table>

14. Formos etiketes (Labels)

Koncepcija: label ir input ryšys (for ir id).
Uzduotis: Sukurkite label „El. paštas“ ir input lauka. Susiekite juos taip, kad paspaudus ant žodžio, žymeklis atsirastų laukelyje.
Testas: Paspaudus ant teksto (label), input laukas turi tapti aktyvus.
CSS
label {
  display: block;
  margin-bottom: 5px;
  cursor: pointer;
}

input {
  padding: 8px;
  border: 1px solid #ccc;
}
HTML
<label for="el-pastas">El. paštas:</label>
<input type="email" id="el-pastas">

15. Fono paveikslelis

Koncepcija: background-image.
Uzduotis: Nustatykite body elementui fono paveiksleli (url) ir nurodykite background-size: cover, kad jis uzdengtu visa ekrana.
Testas: Visas puslapio fonas turi buti paveikslelis, o ne balta spalva.

Sveiki atvyke

CSS
body {
  background-image: url('fonas.jpg');
  background-size: cover;
  background-position: center;
}
HTML
<body>
  <h1>Sveiki atvyke</h1>
</body>
